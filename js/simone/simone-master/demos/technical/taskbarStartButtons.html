Start buttons are created automatically when start menus are found inside a taskbar.

Start menus should have structure compatible with jQuery UI Menu, so ul > li > a for version 1.10, and either that or ul > li for version 1.11+.

Start menus can have multiple levels.

Change language using language select button to see how menus change.
