Introduction

This page describes planned features for Simone. There is no fixed time schedule for those features. You can help make this happen by contributing code to Simone.

Not all of the planned features will land in the declared releases. Some will not be introduced at all, as they may prove unpossible to write right or not worth the efort to implement and maintain.

This roadmap can change anytime. You can search for already implemented features in the changelog.

0.2.0

  • Taskbar: tooltipClose, tooltipOpen events
  • Taskbar: tests for separators
  • Window: embeddedContent: "auto" value
  • Window: more options debug
  • Demos: add possibility to pick jQuery and jQuery UI versions
  • Debug: detecting unsupported jQuery and jQuery UI versions

0.3.0

  • Taskbar: "restoreAll" method connected to maximizeAll button
  • Taskbar: native notification button with connected tooltip
  • Window: drop dependency on jQuery UI Dialog
  • Window: custom titlebar buttons
  • Core: create common widget that window and taskbar will extend

Unspecified future releases

  • Taskbar: windowsZIndexStep option
  • Taskbar: utility to position custom elements connected to custom buttons
  • Taskbar: split resizable/autohide handle into separate elements
  • Window: always on top
  • Window: horizontal/vertical maximization by double-clicking on window edge
  • Docs: create page with major rejected feature ideas
