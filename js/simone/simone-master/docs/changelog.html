Introduction

This page describes changes in Simone. For list of planned features, see the roadmap.

0.1.9

  • Libs: Updated jQuery to 1.11.2 and 2.1.3

0.1.8

  • Taskbar: better recognition of empty windows group.
  • Window: improved handling of global click tracking.

0.1.7

  • Window: fixed active state keeping when window was opened via click coming outside of taskbar/window widgets.

0.1.6

  • Taskbar: compatibility with Ember.js fixed.
  • Taskbar: fixed window button state for group with 1 item.
  • French i18n file added.
  • Window: buttonpane position fixed.
  • Window: jumps while resizing fixed for when top window scroll is present.

0.1.5

  • Tests: jQuery UI 1.11.2 added.
  • Compatibility with jQuery UI 1.11.2 confirmed.
  • Tests: fixed jQuery/jQuery UI propagation to iframes.

0.1.4

  • Spanish i18n file added.
  • Documentation: minor spelling fixes.

0.1.3

  • Dutch i18n file added.
  • Taskbar: fixed text-align on vertical taskbar menu items.

0.1.2

  • German i18n file added.
  • Taskbar: fixed bug with taskbar overlapping opened elements of other taskbars.

0.1.1

  • Fixed typos in links in simone.jquery.json.

0.1.0

  • Initial release.
